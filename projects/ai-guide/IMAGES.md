## DeepSeek 图片描述 Prompt

如果需要用 DeepSeek API 为网站生成配图描述，可复制以下 Prompt：

```
你是一位 AI 配图设计师。请为 "AI Guide - 人工智能学习导航" 网站生成 3 条配图描述，用于在页面中展示。
所有配图使用纯 CSS/SVG 实现，不需要真实图片文件。

配图1：Hero 区域
风格：科技感背景，AI 主题
配色：从 #667eea 到 #764ba2 的渐变
元素：神经网络节点、数据流线

配图2：学习路线图
风格：信息图风格，步骤导向
元素：6个步骤串联，每个步骤有图标

配图3：资源卡片
风格：极简卡片，4种分类色标
元素：圆角卡片，带彩色标签
请用 HTML/CSS 或 SVG 格式输出。
```
